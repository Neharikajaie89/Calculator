# 🧮 Calculator

A simple, responsive calculator built to perform basic arithmetic operations with a clean and user-friendly interface. This project demonstrates fundamental programming concepts, DOM manipulation, and responsive web design.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear all input
- ⌫ Delete last digit
- 🔢 Decimal number support
- 📱 Responsive design for desktop and mobile

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator.git
```

2. Navigate to the project directory:

```bash
cd calculator
```

3. Open `index.html` in your browser.

Or, if using a development server:

```bash
npm install
npm start
```

## 🎯 Usage

1. Enter numbers using the calculator buttons.
2. Choose an arithmetic operator (+, −, ×, ÷).
3. Press `=` to calculate the result.
4. Use `C` to clear all inputs or `DEL` to remove the last digit.
